
# Client-Movement

Fabric mod for Minecraft 26.1 making sprinting and poses entirely client-sided with no server interference. This greatly reduces the amount of movement glitches and inconsistencies for players with high ping and gets rid of the ancient FOV-wobble glitch (MC-4098).
___

This mod could possibly get flagged by certain AntiCheats and might be seen by some as an unfair advantage (you don't lose sprint from hitting entities anymore, but that effect is very minor compared to the 0.6x slowdown applied by the client and purely visual when holding sprint and doesn't apply for PvP), so use at your own discretion.
